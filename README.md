🥖 **Sistema de Gerenciamento - Hamburgueria Online**

Este é um projeto em Python desenvolvido para simular o atendimento de uma hamburgueria online, permitindo que clientes visualizem o cardápio, façam pedidos e escolham a forma de pagamento de maneira simples e interativa via terminal.

O programa utiliza estruturas básicas de programação para controlar o fluxo do menu, entrada de dados e confirmação de pedidos.

---

🚀 **Funcionalidades**

O sistema está dividido em módulos principais:

**1- Acesso do Cliente:**
O sistema solicita o nome do cliente e realiza uma saudação personalizada.

**2- Menu Interativo:**
Interface simples com opções de navegação: Cardápio, Fazer Pedido, Atendimento e Sair.

**3- Cardápio Digital:**
Exibição de produtos como hambúrgueres, porções e bebidas com seus respectivos preços.

**4- Sistema de Pedido:**
Permite ao cliente selecionar itens pelo número, definir quantidade e visualizar sua escolha.

**5- Confirmação de Pedido:**
O cliente pode confirmar ou cancelar o pedido antes de prosseguir.

**6- Formas de Pagamento:**
Suporte para:

* Cartão de Crédito
* Cartão de Débito
* PIX
* Dinheiro

**7- Atendimento ao Cliente:**
Espaço para o usuário digitar dúvidas ou comentários.

---

🛠️ **Tecnologias e Estruturas Utilizadas**

O projeto foi construído utilizando conceitos fundamentais de Python:

* **Entrada de Dados (`input`)**: Interação com o usuário.
* **Saída de Dados (`print`)**: Exibição de mensagens e menus.
* **Laços de Repetição (`while`)**: Mantêm o sistema em execução contínua até o usuário sair.
* **Condicionais (`if/elif/else`)**: Controlam o fluxo do programa e as decisões do usuário.
* **Listas (`[]`)**: Utilizadas para exibir opções de pagamento.
* **f-strings**: Personalização de mensagens com variáveis.

📋 **Como Executar**

1. Certifique-se de ter o Python 3.x instalado.
2. Salve o código em um arquivo chamado:
   `hamburgueria.py`
3. Abra o terminal ou prompt de comando.
4. Execute o comando:

   ```bash
   python hamburgueria.py
   ```

---

📖 **Exemplo de Uso**

Ao iniciar o programa, o usuário seguirá este fluxo:

* **Identificação:** O sistema solicita o nome do cliente.
* **Menu:** O cliente escolhe a opção "1" para ver o cardápio.
* **Pedido:** Seleciona itens como "1" (Hambúrguer Clássico) e define a quantidade.
* **Confirmação:** Confirma o pedido digitando "sim".
* **Pagamento:** Escolhe, por exemplo, "PIX" como forma de pagamento.
* **Finalização:** O sistema informa que o pedido foi confirmado e será processado.
